Chapter 5 - Objects

Objects are a collection of key-value pairs placed inside curly braces {}. Objects have properties that can be any JavaScript value. If it's a function, it's known as a method. An object's properties and methods can be accessed using either dot notation or square bracket notation

Chapter 6 - The Document Object Model (DOM)

The Document Object Model is a way of representing a page of HTML as a tree of nodes. There is a way to access elements on a page using document.getElementById(), document.getElementsByClassName(), document.getElementsByTagNames() and document.querySelector(). The innerHTML property can be used to insert raw HTML directly into the DOM. Manipulating the DOM is a good way to get rid of the alerts.

Chapter 7 - Events

Events occur when a user interacts with a web page. An event listener is attached to an element, then invokes a callback function when the event occurs. You can remove an event using the removeEventListener method.
