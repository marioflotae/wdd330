Chapter 8: Forms

Forms are the primary method used for entering data into a browser. When a form was submitted, it would be sent to a server where the information would be processed using a 'back end' language. Input fields are the most common types of form control, text input, password input, checkbox, etc. document.forms will return an HTML collection of all the forms on a page. Forms also have a submit event that can be used to intercept a form before it's been submitted. The information entered into a form can be read or updated using the value property of the form controls.

Chapter 12 - Object-Oriented Programming in JavaScript

Object-oriented programming (OOP for short) is a style of programming that involves separating the code into objects that have properties and methods. The objects can also be reused or easily modified, as required. There are three main concepts in OOP: encapsulation, polymorphism and inheritance. JavaScript contains a number of built-in constructor functions such as Object, Array, and Function that can be used to create objects, arrays and functions instead of literals. The static keyword can be used in class declarations to create a static method. A static method is called by the class directly rather than by instances of the class. By default, an object's methods are public in JavaScript. Methods can be chained together and called in sequence if they return a reference to this. Methods can be borrowed from other objects.

Chapter 15 - Modern Javascript Development

A JavaScript library is a piece of code that provides several methods that make it easier to achieve common tasks. A good example of how libraries can help save time is in DOM manipulation. Query is the most popular of all the JavaScript libraries used today. Underscore and Lodash are very similar libraries of functions that provide additional functionality to the language. A big advantage of utilizing a popular library is that it will be used by lots of people and thoroughly tested. There are some disadvantages to using libraries, however. You need to include the code for the library as well as your own code. This increases the amount of code that needs to be downloaded by a website, which in some cases can cause performance issues. A module is a self-contained piece of code that provides functions and methods that can then be used in other files and by other modules. Model-View-Controller (MVC) is a design pattern that's been used for a long time in server-side languages. It's a common way of designing software, and used by server-side frameworks such as Ruby On Rails and Django.
